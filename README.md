# qihai.github.io

一个用于 GitHub Pages 的简洁现代静态前端项目。

## 快速开始

1. 在 GitHub 创建一个空仓库，仓库名为 `qihai.github.io`。
2. 在当前目录执行：

```bash
git remote add origin https://github.com/<你的用户名>/qihai.github.io.git
git push -u origin main
```

3. 在 GitHub 仓库设置中，将 **Pages** 的来源设置为 **GitHub Actions**。
4. 工作流执行成功后，你的网站地址为：

`https://qihai.github.io/`

## 本地预览

直接用浏览器打开 `index.html` 即可。
