# qihai.github.io

A minimal modern static frontend project for GitHub Pages.

## Quick start

1. Create an empty GitHub repository named `qihai.github.io`.
2. In this folder run:

```bash
git remote add origin https://github.com/<your-name>/qihai.github.io.git
git push -u origin main
```

3. On GitHub repo settings, set **Pages** source to **GitHub Actions**.
4. After workflow succeeds, your site will be live at:

`https://qihai.github.io/`

## Local preview

Open `index.html` directly in browser.
